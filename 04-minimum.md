# Minimum

Write a function min that takes two arguments and returns their minimum.

function min(a, b) {
  if (a < b) {
    return a;
  } else {
    return b;
  }
}
console.log(min(5, 3)); 
console.log(min(7, 10)); 
console.log(min(5, 5)); 

```
console.log(min(0, 10));
// → 0
console.log(min(0, -10));
// → -10

```
